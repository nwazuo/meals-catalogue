# Prdm Meal Catalogue

## Tasty and healthy meals powered by frictionless collaboration. 

Prdm meal catalogue is a web app for browsing through tasty and healthy meals. You can browse through our catalogue of meals, filter them by category and choose to see only meals from a certain area (e.g. if you want to see only Indian meals üôÇ, we got ya with our filters‚úå). 

Click on any meal to view more details, including ingredients, preparation instructions, etc. 

## Development Setup

This is a Vite + React app. Replace `yarn` with your preferred package manager(npm, pnpm, etc)

- Clone repository

  ```
  git clone https://github.com/nwazuo/meals-catalogue
  ```

* Install dependencies

  ```
  yarn
  ```

* Run development server

  ```
  yarn dev
  ```

## Build Setup

Run build command: 

```
yarn build
```

Preview build: 

```
yarn preview
```
