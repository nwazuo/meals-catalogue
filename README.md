## Development Setup

This is a Vite + React app. Replace `yarn` with your preferred package manager(npm, pnpm, etc)

- Clone repository

  ```
  git clone https://github.com/nwazuo/meals-catalogue
  ```

* Install dependencies

  ```
  yarn
  ```

* Run development server

  ```
  yarn dev
  ```

## Build Setup

Run build command:

```
yarn build
```

Preview build:

```
yarn preview
```
